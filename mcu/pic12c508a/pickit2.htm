PIC12C508A

燒錄程式(PICKit2)

VPP、VDD、GND、DAT、CLK


步驟如下： 

$ cd
$ wget https://github.com/steward-fu/pic12c508/releases/download/v1.0/pickit2.7z
$ 7za x pickit2.7z
$ cd pickit2/pk2cmd/pk2cmd
$ make clean
$ make linux -j4
$ sudo apt-get install libusb-dev libusb-1.0-0-dev libudev-dev -y
$ sudo ./pk2cmd -PPIC12F508 -M -F test.hex
