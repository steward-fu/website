新增VHDL檔案



程式碼如下： 

library ieee;
use ieee.std_logic_1164.all;

entity main is
  port(
    btn: in bit;
    led: out bit
  );
end main;

architecture logic of main is
begin
  led<= btn;
end logic;

相較於Verilog語言，VHDL會比較嚴謹且語法比較偏向Pascal語言，上例就是把btn輸入當做led輸出
