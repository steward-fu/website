STC15W204S

使用stcgal燒録程式(N900)

開發板


Easy Debian環境


USB轉TTL RS-232(PL2303)


準備完成後，連接到N900並切換成USB Host Mode



由於司徒是使用stcgal程式，因此還需要安裝python3、python3-setuptools套件： 

$ sudo apt-get install python3 python3-setuptools python3-serial

接著下載stcgal程式並安裝 

$ cd
$ git clone https://github.com/grigorig/stcgal.git
$ cd stcgal
$ ./setup.py build
$ sudo ./setup.py install

P.S. 司徒發現最新版stcgal過於依賴新的Python3套件，因此，建議下載舊版stcgal，如：stcgal-v1.1

首先査看PL2303裝置使否可正常抓到


執行燒録指令： 

$ sudo stcgal -P stc15 main.hex



執行到此命令時，重新Reset開發板
