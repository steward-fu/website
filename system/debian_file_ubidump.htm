系統 - Debian - File System - 如何解開ubifs檔案


參考資訊：
https://github.com/nlitsme/ubidump

步驟如下： 

$ su
# apt install liblzo2-dev

# cd
# git clone https://github.com/nlitsme/ubidump
# cd ubidump
# pip install -r requirements.txt

# file ../rootfs.img 
    ../rootfs.img: UBIfs image, sequence number 308142, length 4096, CRC 0x9d02e079

# python ./ubidump.py ../rootfs.img --savedir rootfs
# ls rootfs/raw/
    bin  boot  dev  etc  home  lib  linuxrc  media  mnt  proc  sbin  sys  tmp  usr  var
