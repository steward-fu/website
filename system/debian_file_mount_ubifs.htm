系統 - Debian - File System - 如何掛載ubifs檔案


Mount 

# apt-get install mtd-utils

# modprobe ubifs
# modprobe mtdram total_size=256000

# cat /proc/mtd
# dd if=xxx.img of=/dev/mtd0
# ubiattach /dev/ubi_ctrl -m 0
# mount -t ubifs ubi0_0 /mnt

Unmount 

# umount /mnt
# ubidetach /dev/ubi_ctrl -m 0
# rmmod mtdram
