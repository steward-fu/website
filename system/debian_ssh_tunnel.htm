系統 - Debian - SSH Tunnel


參考資料：
https://johnliu55.tw/ssh-tunnel.html

建立Tunnel(執行指令後，會提示使用SSH進Server) 

$ ssh -L 127.0.0.1:9999:192.168.1.27:22 xxx@server_ip -N -f

情境：Local只能連到Server且只有Server可以連到Target 192.168.1.27(Port 22)

Local連接到Target(192.168.1.27) 

$ ssh -p yyy@127.0.0.1:9999

P.S. yyy是Target的帳號
