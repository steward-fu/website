安裝步驟： 

$ source wrsdk-vxworks7-qemu/sdkenv.sh
$ cd ~/wrsdk-vxworks7-qemu/examples/dkm/hello_cmake_dkm
$ mkdir build
$ cd build
$ cmake -D CMAKE_TOOLCHAIN_FILE=~/wrsdk-vxworks7-qemu/vxsdk/sysroot/mk/dkm.toolchain.cmake ..
$ make
