系統 - Docker - 解決Temporary Error: Error configuring auth on host: OS type not recognized問題


參考資訊：
https://github.com/kubernetes/minikube/issues/3952

問題如下： 

$ minikube start
    There is a newer version of minikube available (v1.28.0).  Download it here:
    https://github.com/kubernetes/minikube/releases/tag/v1.28.0

    To disable this notification, run the following:
    minikube config set WantUpdateNotification false
    Starting local Kubernetes v1.10.0 cluster...
    Starting VM...
    E1108 01:30:51.582777   57702 start.go:174] Error starting host: Temporary Error: Error configuring auth on host: OS type not recognized.

解法如下： 

$ minikube delete
$ minikube start --vm-driver=none
