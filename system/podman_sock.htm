系統 - Podman - Podman.sock


參考資訊：
https://docs.podman.io/en/latest/_static/api.html

Info 

$ sudo curl --unix-socket /run/podman/podman.sock http://d/v4.0.0/libpod/info

Pull quay.io/containers/podman 

$ sudo curl -XPOST --unix-socket /run/podman/podman.sock -v 'http://d/v4.0.0/images/create?fromImage=quay.io%2Fcontainers%2Fpodman'

List images 

$ sudo curl --unix-socket /run/podman/podman.sock -v 'http://d/v4.0.0/libpod/images/json' | jq
