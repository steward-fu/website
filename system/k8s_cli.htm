List Pod 

$ sudo kubectl get pod

Remove All Resources 

$ sudo kubectl delete -f xxx.yaml
$ sudo kubectl delete pods --all

Attach Console 

$ kubectl exec --stdin --tty shell-demo -- /bin/bash
