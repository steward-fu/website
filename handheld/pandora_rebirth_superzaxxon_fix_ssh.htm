掌機 - Pandora(Rebirth) - SuperZaxxon - 解決no matching host key type found. Their offer: ssh-rsa,ssh-dss問題


參考資訊：
https://blog.alanwei.com/blog/2022/01/24/ssh-no-matching-host-key-type-found/

問題如下： 

$ ssh steward@10.0.1.3
    Unable to negotiate with 10.0.1.3 port 22: no matching host key type found. Their offer: ssh-rsa,ssh-dss

解法如下： 

$ ssh -o HostKeyAlgorithms=+ssh-rsa -o PubkeyAcceptedKeyTypes=+ssh-rsa steward@10.0.1.3
