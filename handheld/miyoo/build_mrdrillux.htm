Miyoo

build mrdrillux

編譯步驟如下： 

$ cd
$ wget https://github.com/steward-fu/miyoo/releases/download/v1.0/toolchain.7z
$ 7za x toolchain.7z
$ sudo cp -a miyoo /opt/
$ export PATH=$PATH:/opt/miyoo/bin
$ export TARGETMACH=arm-linux
$ export BUILDMACH=i686-pc-linux-gnu
$ export CROSS=arm-linux
$ export CC=${CROSS}-gcc
$ export LD=${CROSS}-ld
$ export AS=${CROSS}-as
$ export CXX=${CROSS}-g++

$ wget https://github.com/steward-fu/miyoo/releases/download/v1.2/mrdrillux.7z
$ 7za x mrdrillux.7z
$ cd mrdrillux
$ make clean
$ make
