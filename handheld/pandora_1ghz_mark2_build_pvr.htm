掌機 - Pandora(1GHz) - Debian(Mark2) - Build GFX_Linux_KM 4.10.00(pvrsrvkm.ko)


步驟如下： 

$ cd
$ git clone git://git.openpandora.org/pandora-kernel.git
$ cd pandora-kernel
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- make omap3_pandora_defconfig
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- make uImage modules
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- INSTALL_MOD_PATH=out make modules_install

$ cd
$ wget https://github.com/steward-fu/website/releases/download/pandora/src_GFX_Linux_KM_4.10.00.tar.gz
$ tar xvf src_GFX_Linux_KM_4.10.00.tar.gz
$ cd GFX_Linux_KM
$ export KERNELDIR=~/pandora-kernel
$ export OMAPES=3.x
$ export TI_PLATFORM=omap3
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- make clean
$ ARCH=arm CROSS_COMPILE=arm-linux-gnueabihf- make
