掌機 - Pandora(Rebirth) - SuperZaxxon - OpenGL ES 2.0 - 開發環境


參考資訊：
https://pandorawiki.org/GLES

步驟如下： 

$ sudo /usr/pandora/scripts/pnd_run.sh -p /media/GAME/pandora/menu/codeblocks.pnd -m

$ export DISPLAY=:0
$ export PATH=/mnt/utmp/codeblocks/usr/bin:$PATH
$ export LD_LIBRARY_PATH=/mnt/utmp/codeblocks/usr/lib:$LD_LIBRARY_PATH

$ gcc main.c -o main -I/mnt/utmp/codeblocks/usr/include/SDL2 -lSDL2 -lGLESv2
$ op_runfbapp ./main

$ sudo /usr/pandora/scripts/pnd_run.sh -p /media/GAME/pandora/menu/codeblocks.pnd -u
