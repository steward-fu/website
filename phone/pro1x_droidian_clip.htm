手機 - F(x)tec Pro1 X - Droidian - Waydroid - 解決Failed to start Clipboard manager service問題


參考資訊：
https://github.com/waydroid/waydroid/issues/396

問題如下： 

$ waydroid session start
    [10:12:49] Failed to start Clipboard manager service, check logs

解法如下： 

$ sudo apt-get install python3-pip
$ pip install pyclip
$ sudo apt install wl-clipboard
