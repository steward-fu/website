手機 - Nokia N900 - Maemo - Build U-Boot


步驟如下： 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/n900/kernel_toolchain_x86.tar.gz
$ wget https://github.com/steward-fu/website/releases/download/n900/src_uboot.tar.gz

$ tar xvf kernel_toolchain_x86.tar.gz
$ sudo mv n900 /opt
$ export PATH=/opt/n900/bin/:$PATH

$ tar xvf src_uboot.tar.gz
$ cd uboot
$ ARCH=arm CROSS_COMPILE=arm-none-linux-gnueabi- make nokia_rx51_defconfig
$ ARCH=arm CROSS_COMPILE=arm-none-linux-gnueabi- make -j4
