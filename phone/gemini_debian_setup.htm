如果要直接從Termux進入Debian，可以使用如下腳本： 

$ vim /data/data/com.termux/files/usr/bin/cli
    #!/data/data/com.termux/files/usr/bin/sh
    tsudo /data/data/ru.meefik.linuxdeploy/files/bin/linuxdeploy -p linux start -m
    ssh xxx@127.0.0.1
    tsudo /data/data/ru.meefik.linuxdeploy/files/bin/linuxdeploy -p linux stop -u

這樣只要在Termux下cli就可以進入Debian，然後，退出時自動執行umount動作
