手機 - Nokia N900 - PostMarketOS - 安裝系統


參考資訊：
https://wiki.postmarketos.org/wiki/Nokia_N900
https://github.com/postmarketOS/pmbootstrap/releases
https://wiki.bibanon.org/Installing_postmarketOS_on_the_Nokia_N900

步驟如下： 

$ cd
$ sudo apt-get install python3.0 qemu-user-static
$ git clone https://gitlab.com/postmarketOS/pmbootstrap
$ cd pmbootstrap
$ ./pmbootstrap.py init

P.S. Work路徑必須位於$HOME/.local/var/pmbootstrap

Maemon 

$ sudo vim /etc/bootmenu.d/10-pmos.item
    ITEM_NAME="postmarketOS"
    ITEM_SCRIPT="boot.scr"
    ITEM_DEVICE="${EXT_CARD}p1"
    ITEM_FSTYPE="ext2"

$ u-boot-update-bootmenu

完成
