步驟如下：
1. Enable USB Debugging
2. Enable USB ADB Access
3. 執行如下命令 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/xt894/saferoot.zip
$ unzip saferoot.zip
$ cd saferoot
$ sh ./install.sh
