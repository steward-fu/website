手機 - Nokia N900 - Maemo - 解決PSX4M check permission to keys file問題


雖然N900上有PSX4M模擬器，不過在設定完Keys時，竟出現Somethings goes wrong, check permission to keys file訊息，如下圖：


該訊息代表無足夠權限修改檔案，解法如下： 

$ sudo mkdir -p /home/opt/psx4m/
$ sudo chmod 0777 /home/opt/psx4m/
$ sudo rm /home/opt/psx4m/keys
$ sudo chown -R user:users /home/opt/psx4m/
$ sudo chmod 0777 /home/user/.config/Team16/
$ sudo chown -R user:users /home/user/.config/Team16/
