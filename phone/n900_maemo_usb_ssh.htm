手機 - Nokia N900 - Maemo - 如何透過USB連接SSH


參考資訊：
https://blog.alanwei.com/blog/2022/01/24/ssh-no-matching-host-key-type-found/

步驟如下：
1. 連接USB到PC
2. 選擇PC-Suite Mode
3. Maemo執行如下命令 

$ sudo ifconfig usb0 192.168.56.100 netmask 255.255.255.0

4. PC執行如下命令 

$ sudo ifconfig usb0 192.168.56.101 netmask 255.255.255.0
$ ssh -o HostKeyAlgorithms=+ssh-rsa -o PubkeyAcceptedKeyTypes=+ssh-rsa user@192.168.56.100
