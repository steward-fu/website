手機 - Motorola XT897 - Sailfish OS - 解決High cpu usage: tracker-miner-fs問題


外部MicroSD如果檔案太多，tracker daemon將會一直佔著CPU，目前解法只能rename tracker process

步驟如下： 

$ cd /usr/libexec/
$ sudo mv tracker-extract __tracker-extract
$ sudo mv tracker-miner-fs __tracker-miner-fs
$ sudo mv tracker-store __tracker-store
$ sudo mv tracker-writeback __tracker-writeback
