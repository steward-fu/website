手機 - Cosmo Communicator - Android - 如何手動更新Cover Display韌體


參考資訊：
https://support.planetcom.co.uk/index.php/Cosmo_Firmware_upgrade
http://support.planetcom.co.uk/download/CodiLatest/Cosmo_firmware-stem_ospi2_1_1_1_14.bin
http://support.planetcom.co.uk/download/CodiLatest/Cosmo_resource-stem_resource_1_1_1_14.bin

使用者不用擔心更新CoDi韌體會導致外部顯示器變磚，因為，即使更新失敗，仍然可以再次刷機，只是不見得一次就可以刷成功，CoDi更新步驟如下：
1. 開啟Aeroplane mode、Do not disturb、Cover Dislay
2. 下載Cosmo_firmware-stem_ospi2_1_1_1_14.bin、Cosmo_resource-stem_resource_1_1_1_14.bin並且放到Internal Flash(放在外部MicroSD會導致App Crash)
3. 開啟Cover Display Assistant
4. ADVANCED => Flash Image Manually => Cosmo_firmware-stem_ospi2_1_1_1_14.bin => FLASH NOW
5. ADVANCED => Flash Image Manually => Cosmo_resource-stem_resource_1_1_1_14.bin => FLASH NOW
P.S. 司徒因為使用線上更新失敗，最後才使用手動更新成功，如果一直更新失敗，建議重新開機試試
