手機 - Nokia N900 - Maemo - 連接Dummy Network


為何需要Dummy Network呢？原因在於，如果使用USB轉RJ-45網卡上網時，Maemo系統無法得知目前是否已經連上網路，因此，需要一個假的Dummy Network，用來欺騙Maemo系統，這樣系統就可以正常更新Email、聊天訊息等事項

操作步驟如下： 

$ sudo apt-get install libicd-network-dummy
$ sudo gconftool-2 -s -t string /system/osso/connectivity/IAP/DUMMY/type DUMMY
$ sudo gconftool-2 -s -t string /system/osso/connectivity/IAP/DUMMY/name "Dummy network" 
$ sudo gconftool-2 -s -t string /system/osso/connectivity/IAP/DUMMY/autoconnect false 
$ sudo gconftool -s /system/osso/connectivity/network_type/auto_connect -t list --list-type string "[WLAN_INFRA,GPRS]" 
$ sudo killall icd2
$ sudo reboot

完成
