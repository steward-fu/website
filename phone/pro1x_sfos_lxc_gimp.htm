手機 - F(x)tec Pro1 X - Sailfish OS - LXC - 解決GIMP字型太小的問題


參考資料：
https://gitlab.gnome.org/GNOME/gimp/-/issues/4892

步驟如下： 

$ vim ~/.config/GIMP/2.10/gtkrc
    #To set the font
    style "Monaco"
    {
        font_name = "Monaco 18"
    }
    widget_class "*" style "Monaco"
    gtk-font-name = "Monaco 18"
