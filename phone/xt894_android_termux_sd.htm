手機 - Motorola XT894 - Android - 如何將Termux移到外部MicroSD


參考資訊：
https://github.com/termux/termux-app/discussions/3567

步驟如下：
1. 格式化MicroSD並且設定成Internal Storage
2. Settings => Developer options => Force allow apps on external
3. Settings => Apps => Termux => Storage => Change Storage
4. ADB Command 

$ adb root
$ adb shell
# cd /data/data
# ln -s /mnt/expand/xxx-xxx-xxx/user/0/com.termux
