手機 - Blackberry Passport - Core Native (C/C++) - Build freetype 2.4.0


步驟如下： 

$ cd
$ wget https://download-mirror.savannah.gnu.org/releases/freetype/freetype-2.4.0.tar.gz
$ tar xvf freetype-2.4.0.tar.gz
$ cd freetype-2.4.0
$ source ~/bbndk/bbndk-env_10_3_1_995.sh
$ export TARGETMACH=arm-unknown-nto-qnx8.0.0eabi
$ export BUILDMACH=i686-pc-linux-gnu
$ export CROSS=arm-unknown-nto-qnx8.0.0eabi
$ export CC=${CROSS}-gcc
$ export LD=${CROSS}-ld
$ export AS=${CROSS}-as
$ export CXX=${CROSS}-g++
$ ./configure --host=$TARGETMACH --prefix=$QNX_TARGET
$ make 
$ make install
