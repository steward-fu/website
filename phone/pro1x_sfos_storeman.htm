手機 - F(x)tec Pro1 X - Sailfish OS - 如何安裝Storeman


參考資訊：
https://openrepos.net/content/basil/about-openrepos
https://github.com/storeman-developers/harbour-storeman-installer/releases/tag/2.2.0

步驟如下： 

$ cd
$ wget https://github.com/storeman-developers/harbour-storeman-installer/releases/download/2.2.0/harbour-storeman-installer-2.2.0-release1.noarch.rpm
$ sudo zypper install harbour-storeman-installer-2.2.0-release1.noarch.rpm
