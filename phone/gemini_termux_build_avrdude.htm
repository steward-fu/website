手機 - Gemini PDA 4G - Android - Termux - Build AVEDUDE 6.3


步驟如下： 

$ pkg install readline-dev libelf-dev
$ cd
$ wget http://download.savannah.gnu.org/releases/avrdude/avrdude-6.3.tar.gz
$ tar xvf avrdude-6.3.tar.gz
$ cd avrdude-6.3
$ ./configure --prefix=$PREFIX --build=arm
$ make
$ make install
$ avrdude
