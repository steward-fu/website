手機 - Motorola XT897 - Sailfish OS - 解決Configure: error: no acceptable grep could be found問題


步驟如下： 

$ cd
$ wget https://ftp.gnu.org/gnu/grep/grep-2.0.tar.gz
$ tar xvf grep-2.0.tar.gz
$ cd grep-2.0
$ ./configure
$ make
$ sudo make install

P.S. 原本的grep是busybox symlink，而使用v2.0的原因是，v2.0不會使用grep做configure，因此，安裝完v2.0 grep後，就可以升到高版本grep
