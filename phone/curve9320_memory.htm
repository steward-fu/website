手機 - Blackberry Curve9320 - 自動釋放記憶體


Blackberry APP World上面有許多釋放記憶體的工具，多數基本上是需要付費的，但是，其實這個功能黑莓機本身就已經有支援，所以，知道這個功能的人，基本上就不需要付費去下載安裝，而多數付費下載的記憶體釋放工具，說穿了也只是呼叫系統本身提供的記憶體釋放功能，其實不需要花這種錢的，今天司徒就介紹一下這個功能，如果需要記憶體釋放工具就可以參考司徒的作法，不用花錢去買本身黑莓機就有的功能

開啟記憶體釋放功能(預設是沒有開啟這個功能)：選項 => 安全性 => 進階安全性設定 => 記憶體清理 => 啟用
