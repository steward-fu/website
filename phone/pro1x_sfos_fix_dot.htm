手機 - F(x)tec Pro1 X - Sailfish OS - 解決Program dot not found問題


解法如下：

$ cd
$ wget https://gitlab.com/api/v4/projects/4207231/packages/generic/graphviz-releases/8.0.1/graphviz-8.0.1.tar.gz
$ tar xvf graphviz-8.0.1.tar.gz
$ cd graphviz-8.0.1
$ ./autogen.sh
$ ./configure
$ make -j4
$ sudo make install
