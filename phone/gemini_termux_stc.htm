手機 - Gemini PDA 4G - Android - Termux - 使用stcgal燒錄STC15W4K56S4


開發板


接腳：

STCPL2303
GNDGND
P3.0UART TX
P3.1UART RX



安裝stcgal 

$ cd
$ pkg install libtool automake pkg-config root-repo python
$ git clone https://github.com/grigorig/stcgal
$ cd stcgal
$ ./setup.py build
$ ./setup.py install

main.c 

#include <8051.h>
#include <compiler.h>
SFR(P5, 0xC8);
  
void delay(unsigned long cnt)
{
    while (cnt--);
}
  
void main(void)
{
    while (1) {
        P5 = 0xff;
        delay(1000000);
        P5 = 0x00;
        delay(1000000);
    }
}

編譯 

$ sdcc main.c
