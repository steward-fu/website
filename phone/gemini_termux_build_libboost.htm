手機 - Gemini PDA 4G - Android - Termux - Build libboost 1.61.0


步驟如下： 

$ cd
$ pkg install python-dev libicu g++
$ wget https://sourceforge.net/projects/boost/files/boost/1.61.0/boost_1_61_0.tar.gz
$ tar xvf boost_1_61_0.tar.gz
$ cd boost_1_61_0
$ ./bootstrap.sh --prefix=$PREFIX
$ ./b2
$ cp -a boost /data/data/com.termux/files/usr/include/
$ cp -a stage/lib/* /data/data/com.termux/files/usr/lib/
