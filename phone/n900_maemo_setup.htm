手機 - Nokia N900 - Maemo - 安裝系統


步驟如下：
1. 關機
2. 按住U鍵
3. 連接USB
4. 執行如下命令 

$ cd 
$ wget https://github.com/steward-fu/website/releases/download/n900/firmware.7z
$ 7za x firmware.7z
$ cd firmware
$ ./flasher_debian -k zImage -f -R
$ ./flasher_debian -F RX-51_2009SE_21.2011.38-1_PR_COMBINED_MR0_ARM.bin -f -R
$ ./flasher_debian -F RX-51_2009SE_10.2010.13-2.VANILLA_PR_EMMC_MR0_ARM.bin -f -R
