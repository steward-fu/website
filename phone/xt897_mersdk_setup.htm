手機 - Motorola XT897 - mersdk - 安裝SDK


參考資料：
https://sailfishos.org/wiki/Application_SDK
https://github.com/Karry/osmscout-sailfish/wiki/Howto-build-for-Sailfish-OS

步驟如下： 

$ cd
$ wget https://releases.sailfishos.org/sdk/installers/3.3.6/SailfishSDK-3.3.6-linux64-offline.run
$ chmod +x SailfishSDK-3.3.6-linux64-offline.run
$ ./SailfishSDK-3.3.6-linux64-offline.run

安裝完成後，VirtualBox就會有如下新加入的系統，開啟Sailfish OS Build Engine


系統出現如下畫面後


使用如下命令連接： 

$ ssh mersdk@localhost -p 2222 -i ~/SailfishOS/vmshare/ssh/private_keys/engine/mersdk
