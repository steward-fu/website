2496行做spin_trylock的動作，但是，在msm_fb_pan_idle()卻做了sleep的動作，導致crash

Workaround： 

DEFINE_SEMAPHORE(msm_fb_pan_sem);
static int msm_fb_pan_idle(struct msm_fb_data_type *mfd)
{
    return 0;
}
