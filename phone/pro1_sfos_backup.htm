手機 - F(x)tec Pro1 - Sailfish OS - 如何備份系統


步驟如下：
1. 進入系統備份/dev/block/platform/soc/by-name/boot_a
2. 進入系統備份/dev/block/platform/soc/by-name/boot_b
3. 按住POWER和VOL-進入Fastboot模式 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/pro1/twrp-3.3.1-qx1000.img
$ fastboot flash boot twrp-3.3.1-qx1000.img

4. 進入Recovery模式(先進入Fastboot模式，再透過選單進入Recovery模式)


5. 透過ADB備份 

$ adb pull /dev/block/sda sda.img
    /dev/block/sda: 1 file pulled. 32.3 MB/s (121911640064 bytes in 3594.755s)

$ adb pull /dev/block/sdb sdb.img
    /dev/block/sdb: 1 file pulled. 29.3 MB/s (4194304 bytes in 0.137s)

$ adb pull /dev/block/sdc sdc.img
    /dev/block/sdc: 1 file pulled. 29.0 MB/s (4194304 bytes in 0.138s)

$ adb pull /dev/block/sdd sdd.img
    /dev/block/sdd: 1 file pulled. 32.0 MB/s (134217728 bytes in 3.996s)

$ adb pull /dev/block/sde sde.img
    /dev/block/sde: 1 file pulled. 32.4 MB/s (4294967296 bytes in 126.290s)

$ adb pull /dev/block/sdf sdf.img
    /dev/block/sdf: 1 file pulled. 32.4 MB/s (1610612736 bytes in 47.422s)

6. 再度進入Fastboot模式並且回復boot_a、boot_b 

$ fastboot flash boot_a xxx
$ fastboot flash boot_b xxx
