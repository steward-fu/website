手機 - Nokia N900 - Kali - 安裝系統


步驟如下： 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/n900/kali-rolling.7z
$ 7za x kali-rolling.7z
$ sudo dd if=kali-rolling.img of=/dev/sdX bs=1M

Maemo 

$ sudo vim /etc/bootmenu.d/30-kali.item 
    ITEM_NAME="Kali Linux kali-rolling armhf 4.5.0-rc1+"
    ITEM_DEVICE="\${EXT_CARD}p1"
    ITEM_FSTYPE="ext4"
    ITEM_KERNEL="/boot/uImage-4.5.0-rc1+"
    ITEM_INITRD="/boot/uInitrd-4.5.0-rc1+"
    ITEM_CMDLINE="root=/dev/mmcblk0p1 rootwait ro console=tty0 vram=12M ubi.mtd=5"

$ u-boot-update-bootmenu
