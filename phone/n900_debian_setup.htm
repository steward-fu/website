手機 - Nokia N900 - Debian - 安裝系統


步驟如下： 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/n900/jessie.dd.xz
$ xz -d jessie.dd.xz
$ sudo dd if=jessie.dd of=/dev/sdX bs=1M

Maemo 

$ sudo vim /etc/boot.d/30-jessie.item
    ITEM_NAME="Debian Jessie"
    ITEM_KERNEL="uImage"
    ITEM_INITRD="uInitrd"
    ITEM_DEVICE="${EXT_CARD}p1"
    ITEM_FSTYPE="vfat"
    ITEM_OMAPATAG="1"
    ITEM_CMDLINE="root=/dev/mmcblk0p2 rootwait vram=12M console=tty0 quiet"

$ u-boot-update-bootmenu
