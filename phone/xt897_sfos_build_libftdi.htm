手機 - Motorola XT897 - Sailfish OS - Build libftdi


步驟如下： 

$ sudo zypper install cmake

$ cd
$ wget https://src.fedoraproject.org/lookaside/pkgs/libftdi/libftdi-0.19.tar.gz/md5/e6e25f33b4327b1b7aa1156947da45f3/libftdi-0.19.tar.gz
$ tar xvf libftdi-0.19.tar.gz
$ mkdir -p libftdi-0.19/build
$ cd libftdi-0.19/build
$ cmake ..
$ make
$ sudo make install
