手機 - Nokia N900 - Maemo - Build Kernel (Stock)


步驟如下： 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/n900/kernel_toolchain_x86.tar.gz
$ wget https://github.com/steward-fu/website/releases/download/n900/src_kernel_stock.tar.gz

$ tar xvf kernel_toolchain_x86.tar.gz
$ sudo mv n900 /opt
$ export PATH=/opt/n900/bin/:$PATH

$ tar xvf src_kernel_stock.tar.gz
$ cd kernel
$ ARCH=arm CROSS_COMPILE=arm-none-linux-gnueabi- make rx51_defconfig
$ ARCH=arm CROSS_COMPILE=arm-none-linux-gnueabi- make zImage modules -j4
$ ARCH=arm CROSS_COMPILE=arm-none-linux-gnueabi- INSTALL_MOD_PATH=out make modules_install
