手機 - F(x)tec Pro1 - Sailfish - 安裝系統


參考資料：
https://gitlab.com/sailfishos-porters-ci/t5-ci/-/pipelines?page=1&scope=finished
https://community.fxtec.com/topic/2467-community-build33016updated-18th-aug-sailfish-os-for-fxtec-pro1/

步驟如下：
1. 按住POWER和VOL-進入Fastboot模式
2. 解鎖Bootloader 

$ sudo fastboot flashing unlock

3. 執行如下命令： 

$ cd
$ wget https://github.com/steward-fu/website/releases/download/pro1/platform-tools_r33.0.3-linux.zip
$ wget https://github.com/steward-fu/website/releases/download/pro1/sailfish.zip

$ unzip platform-tools_r33.0.3-linux.zip
$ sudo mv platform-tools /opt
$ export PATH=/opt/platform-tools:$PATH

$ unzip artifacts.zip
$ unzip sfe-t5-4.4.0.64-devel-20220724/Sailfish_OS--devel-20220724-4.4.0.64-t5-0.0.1.50.zip
$ cd Sailfish_OS--devel-20220724-4.4.0.64-t5-0.0.1.50
$ ./flash.sh
$ fastboot reboot

完成

P.S. 如果卡在Encrypting user data, please wait畫面5分鐘以上，可以直接重新開機
