手機 - Motorola XT894 - Android - 安裝系統


參考資訊：
https://xdaforums.com/t/recovery-safestrap-3-75-for-unused-partitions-preinstall-webtop.3023932/
https://www.internauta37.altervista.org/en/xt894-and-xt912-safestrap-375-unused-partitions-preinstall-webtop

步驟如下：

$ cd
$ wget https://github.com/steward-fu/website/releases/download/xt894/VRZ_XT894_9.8.2O-72_VZW-18-8_CFC.xml.zip
$ unzip VRZ_XT894_9.8.2O-72_VZW-18-8_CFC.xml.zip
$ cd VRZ_XT894_9.8.2O-72_VZW-18-8_CFC.xml
$ sudo fastboot flash cdt.bin cdt.bin_patch
$ sudo fastboot reboot-bootloader
$ sudo fastboot flash emstorage emstorage.img
$ sudo fastboot reboot-bootloader
$ sudo fastboot flash mbm allow-mbmloader-flashing-mbm.bin
$ sudo fastboot reboot-bootloader
$ sudo fastboot flash mbmloader mbmloader.bin
$ sudo fastboot flash mbm mbm.bin
$ sudo fastboot oem fb_mode_set
$ sudo fastboot reboot-bootloader
$ sudo fastboot flash cdt.bin cdt.bin
$ sudo fastboot reboot-bootloader
$ sudo fastboot erase cache
$ sudo fastboot erase userdata
$ sudo fastboot flash logo.bin logo.bin
$ sudo fastboot flash ebr ebr
$ sudo fastboot flash mbr mbr
$ sudo fastboot flash devtree device_tree.bin
$ sudo fastboot flash boot boot.img
$ sudo fastboot flash system system.img
$ sudo fastboot flash recovery recovery.img
$ sudo fastboot flash cdrom cdrom
$ sudo fastboot flash preinstall preinstall.img
$ sudo fastboot flash radio radio.img
$ sudo fastboot oem fb_mode_clear
$ sudo fastboot reboot
