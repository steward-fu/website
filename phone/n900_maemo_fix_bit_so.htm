解法如下： 

$ cd
$ wget http://bitop.luajit.org/download/LuaBitOp-1.0.2.tar.gz
$ tar xvf LuaBitOp-1.0.2.tar.gz
$ cd LuaBitOp-1.0.2
$ vim Makefile
    6 INCLUDES= -I/usr/include/lua5.1

$ make
$ sudo mkdir -p /usr/local/lib/lua/5.1
$ sudo make install
