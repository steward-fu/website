手機 - Motorola XT897 - Sailfish OS - Build tmux


步驟如下： 

$ sudo zypper install libtool

$ cd
$ git clone https://github.com/libevent/libevent
$ cd libevent
$ ./autogen.sh
$ ./configure
$ make
$ sudo make install

$ cd
$ git clone https://github.com/tmux/tmux
$ cd tmux
$ ./autogen.sh
$ ./configure
$ make
$ sudo make install
