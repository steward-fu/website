hello, world!永遠都是作為第一個教學範例，除了可以知道基本程式的框架之外，最主要的原因在於熟悉如何編譯和執行程式，司徒將以這樣的流程來介紹如何撰寫一個hello, world!驅動程式，而在此涉獵的Linux指令則必須依賴使用者先行上網學習，這部份是事前必須準備的東西，接著就先架設編譯環境。

$ cd

ldd.S(ldd.S如果改成ldd.s，則會被make clean刪掉，請注意) 

.global init_module
.global cleanup_module
.extern printk

.section .modinfo, "ax"
__mod_kernel_version: .asciz "kernel_version=4.1.18";    .align
__mod_license:        .asciz "license=GPL";              .align
__mod_author:         .asciz "author=Steward_Fu";        .align
__mod_description:    .asciz "description=Linux Driver"; .align
 
.section .text
msg_load:   .asciz "hello, world!\n"; .align
msg_unload: .asciz "unload it\n";     .align
 
.section .text
cleanup_module:
  push {lr}
  ldr r0, =msg_unload
  bl printk
  pop {pc}

init_module:
  push {lr}
  ldr r0, =msg_load
  bl printk
  mov r0, #0
  pop {pc}
.end

init_module()掛載驅動程式時，系統執行呼叫的副程式，目前僅列印訊息，其餘事情不處理。
cleanup_module()卸載驅動程式時，系統呼叫的副程式，目前僅列印訊息，其餘事情不處理。

Makefile 

export ARCH=arm
export CROSS_COMPILE=arm-linux-gnueabihf-
export AS=${CROSS_COMPILE}as
KERNEL=$(HOME)/kernel

obj-m += main.o
main-objs:= ldd.o
all:
  $(AS) -o ldd.o ldd.S
  make -C $(KERNEL) M=$(PWD) modules

clean:
  make -C $(KERNEL) M=$(PWD) clean

接著輸入make指令編譯就可以產生main.ko

把main.ko複製到第一個FAT磁區，接著插入SDCard，開機執行如下指令即可： 

# cd /boot
# ls
MLO             main.ko         zImage
am335x-evm.dtb  u-boot.img

# insmod main.ko
[   25.472901] hello, world!

如果要卸載驅動程式，只要輸入如下指令即可 

# rmmod main
[   73.074343] unload it
