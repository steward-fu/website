Sharp Netwalker PC-Z1

安裝Android 4.0.3(MicroSD)

由於熱愛Netwalker PC-Z1機器的高手相當多，因此，新版的Android 4.0.3系統也同樣被移植到Netwalker PC-Z1機器上，整體的操作步驟都跟Android 2.3.4系統一樣，差別僅在於下載的檔案不一樣，操作步驟說明如下。

安裝步驟
1. 製作Android 4.0.3系統 

# wget http://mit.sharp.co.jp/os.tar.bz2
# mkdir -p /media/rootfs
# tar xf os.tar.bz2 -C /media/rootfs
# echo "" > /media/rootfs/etc/udev/rules.d/70-persistent-net.rules
# mv /media/rootfs/etc/rc2.d/S01upfirm /media/rootfs/etc/rc2.d/~S01upfirm
# vim /media/rootfs/boot/boot.conf
Edit the console option in /media/rootfs/boot/boot.conf as console=tty1

# wget http://brain.cc.kogakuin.ac.jp/research/files/ics/zImage
# cp zImage /media/rootfs/boot/
# mkdir -p /media/rootfs/android
# wget http://brain.cc.kogakuin.ac.jp/research/files/ics/ics4.0.3-20120102.tgz
# tar zxf ics4.0.3-20120102.tgz -C /media/rootfs/android
# vim /media/rootfs/etc/rc.local
Edit the /media/rootfs/etc/rc.local, and put a line /usr/sbin/chroot /android/ics /init before "exit 0"

2. 使用gparted指令將4GB MicroSD(不要使用高速卡)分割成2GB ext3、512MB linux-swap、1.5GB fat32分割表
3. 將/media/rootfs資料夾裡面的全部資料(包含資料夾)複製到4GB MicroSD的2GB ext3分割表的根目錄
4. 按住Netwalker PC-Z1機器的滑鼠左鍵 + 滑鼠右鍵 + 電源鍵(出現Sharp Logo後就可以先放開電源鍵)

P.S. 由於這個Android系統尚未被root過，因此，整體的CPU速度是介於160MHz~800MHZ之間，這樣的速度表現算是有點慢，建議使用者可以嘗試root並使用超頻工具做效能提升的動作。
