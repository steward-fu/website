Sharp Zaurus SL-C700

開機模式

Zaurus機器的第一個開機Bootloader程式是位於NOR Flash Memory，如果電源啟動時沒有按特殊按鍵，則機器就會從NAND Flash Memory的0x0000位址開始執行，Zaurus SL-700~SL-C3200機器都支援特殊按鍵的功能，司徒將特殊按鍵的組合整理成如下表格。

特殊按鍵 功能描述
Fn + Home Power On
Fn + D + P Normal Dialog
Fn + D + M Service Dialog
A + B Backup
D + A Angel Debug Monitor version 0.42(UART 9600bps)
D + B 從MTD1 zImage-Stage2(0x220000)開始執行
D + C Update NAND flash(需要Updator.pro以及各區域BIN檔案)
D + K Update NAND flash(需要Updator.pro以及各區域BIN檔案)
D + L Update NAND flash(需要Updator.pro以及各區域BIN檔案)
D + M Service Dialog
D + P Normal Dialog
D + Q Update NAND flash(需要Updator.pro以及各區域BIN檔案)
D + R Remote Check(UART 2400bps)
D + V Backup
D + Z Update NOR flash(需要Updator.pro以及各區域BIN檔案)
L + X Normal Dialog
Q + T Update NOR flash(需要Updator.pro以及各區域BIN檔案)
